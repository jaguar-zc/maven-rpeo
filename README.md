##### icooding仓库jar包仓库

###### 使用方法
``` xml
	<!-- 项目仓库位置-->
	<repository>
	  <id>my-repository</id>
	  <url>https://raw.githubusercontent.com/jaguar-zc/maven-rpeo/master</url>
	  </repository> 
	<!--项目坐标-->
	<dependency>
		<groupId>com.icooding</groupId>
		<artifactId>icooding-doc</artifactId>
		<version>1.0-SNAPSHOT</version>
	</dependency>
```
